  -Frontend:
    -AngularJS
      -Currently uses ngRoute, I'm going to switch it over to ui-router
    -Bootstrap
    -SASS

    -e2e Test with Protractor

    -Bower - Package Management
    -Gulp/Grunt - Task Manager (*Not Installed Yet)
